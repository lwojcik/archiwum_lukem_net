Mam Windows Home Server

12.08.2008, 14:43 Komputery

Niecały miesiąc temu po wpisie przeciętnego zamówiłem darmową 120-dniową wersję Windows Home Server. Prawie bym o tym zapomniał, gdyby nie dzisiejsza wizyta zagubionego kuriera UPS (który szukał mnie poza domem zamiast dostrzec i wykorzystać numer telefonu na kopercie), który wręczył mi sporych rozmiarów kopertę z trzema płytami DVD.



Dementuję: tak jak widać na załączonych zdjęciach, płyty dotarły nie w “ładnym pudełeczku”, lecz ordynarnych papierowych kopertach. Poza tym nie wiem jeszcze czy i do czego go wykorzystam (w końcu 120 dni zabawy to nie tak znowu mało). Fajnie by było, gdyby można było zrobić z niego system dla stacji roboczej, tak jak, o ile pamiętam, było możliwe z Windows Server 2003 (i chyba nawet z Windows Server 2008?), ale z HS to chyba (czyt.: na pewno) nie przejdzie.
